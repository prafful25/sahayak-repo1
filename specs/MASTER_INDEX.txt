------------------------------------------------------------

— no new domain provisioning, no new Docker container, no new SSL cert.
Every spec reuses existing scraper infrastructure on Phase 1-3 VPS.

          (2) Add new endpoint to existing FastAPI container
          (3) Update Razorpay billing to expose new tier
          (4) Update Redis cache key namespace
